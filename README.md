## 🏢 Sobre Nosotros
Fundada en **1962** por un grupo visionario de 97 trabajadores, la **Sociedad Cooperativa de Producción de Servicios Juárez S.C.L.** es un referente nacional en el autotransporte de materiales de construcción, barrenación y explotación minera. 

Con más de 60 años de trayectoria, nuestra organización se enfoca en el crecimiento laboral y personal de sus integrantes, contribuyendo al desarrollo de infraestructura en todo México bajo los valores de solidaridad y mejora continua.

## 🛠️ Nuestros Servicios
Ofrecemos soluciones integrales para la industria de la construcción e infraestructura:
*   **Suministro de Materiales:** Grava, balastro, piedra, y agregados pétreos.
*   **Logística y Transporte:** Autotransporte especializado y manejo de materiales en obra.
*   **Renta de Maquinaria Pesada:** Bulldozers, motoniveladoras y equipos mineros.
*   **Servicios de Obra:** Barrenación, explosión controlada y suministro de agua potable.
*   **Explotación de Canteras:** Gestión sostenible de recursos naturales.

## 🤝 Grupo Cooperativo (Filiales)
*   **Transportes Hidro Hidalguenses S.A. DE C.V.:** Transporte foráneo y explotación de yacimientos.
*   **Juárez de Oriente S.A. DE C.V.:** Especialistas en logística regional.
*   **Unidad Médica Integral Hidalguense S.A DE C.V.:** Atención médica integral para la comunidad.

## ⚙️ Configuración del Proyecto

### Requisitos Previos
*   Node.js (Versión recomendada: v18 o superior)
*   npm o yarn

### Instalación
```bash
# Clonar el repositorio
git clone [url-del-repositorio]

# Entrar al directorio
cd web_page_cooperativa_juarez_scl

# Instalar dependencias
npm install
```

### Desarrollo
```bash
npm run dev
```

### Construcción para Producción
```bash
npm run build
```
